# Wedding Sites

The static website for my wedding.

**Prerequistes:**

CLI Commands
1.   `bash` (Bourne-Again Shell)
2.   `clang` (C Preprocessor)
3.   `magick` (ImageMagick CLI Tool)
4.   `ninja` (Ninja Build Tool https://ninja-build.org/)
5.   `npx` (Node.js)
6.   `rsvg-convert` (Tool to rasterize SVG files)
7.   `yarn` (JS Package Manager)
8.   `zip` (ZIP Archiving)

**Build:**

```
./haga
```
