# Wedding Sites

The static website for my wedding.

**Prerequistes:**

CLI Commands
1.   `bash` (Bourne-Again Shell)
2.   `clang` (C Preprocessor)
3.   `ninja` (Ninja Build Tool https://ninja-build.org/)
4.   `npx` (Node.js)
5.   `yarn` (JS Package Manager)
6.   `zip` (ZIP Archiving)

**Build:**

```
./haga
```
